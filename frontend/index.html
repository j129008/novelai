Tag 順序

人數 → 角色/系列 → 外觀 → 動作/表情 → 場景 → 風格

1girl, character name, blonde hair, blue eyes, smile, garden, soft lighting

強調語法

{text}加強 ×1.05{red hair}
{{text}}加強 ×1.10{{masterpiece}}
[text]減弱 ×0.95[background]
1.5::text::精確權重1.5::rain, night::
-1::text::反向移除-1::monochrome::

多角色

用 | 分隔 base prompt 和各角色：

base prompt | 角色1描述 | 角色2描述

Base prompt 放什麼：

  • 人數：2girls、1boy, 1girl
  • 場景/背景：park、classroom、sunset
  • 風格/氛圍：watercolor、dramatic lighting、year 2024
  • 構圖：from above、wide shot、close-up

角色 prompt 放什麼：

  • 只寫 girl/boy（不加數字）
  • 外觀：blonde hair, blue eyes, school uniform
  • 表情/動作：smile, waving
  • 角色名：methode (sousou no frieren)

完整範例：

2girls, cherry blossoms, spring, sunny, watercolor | girl, blonde hair, white dress, smile, waving | girl, black hair, blue uniform, holding book, calm

互動標籤：source#hug（發起）、target#hug（接受）、mutual#holding hands（雙向）

排列順序 = 畫面位置：第一個角色在左，第二個在右，依此類推

特殊標籤

fur dataset,放最前面，切換毛茸茸風格
background dataset,放最前面，純風景/靜物（無人物）
year 2024調整畫風年代感

Quality Tags（自動加入）

開啟時自動在 prompt 末尾加：

, location, very aesthetic, masterpiece, no text

範例

角色 1girl, long white hair, blue eyes, elegant dress, sitting, garden, roses, soft lighting
動態 1boy, action pose, sword, jumping, ruined castle, dramatic lighting, cape flowing
風景 background dataset, mountain lake, sunrise, fog, pine trees, reflection
多角色 2girls, cafe, warm lighting, cozy | girl, short pink hair, cheerful, waving | girl, glasses, reading book, calm
多角色互動 1boy, 1girl, sunset, rooftop, dramatic | boy, black hair, source#hug | girl, red scarf, target#hug, blush

小技巧

  • 越前面的 tag 影響力越強
  • V4.5 支援自然語言描述，不一定要用 tag
  • Undesired Content 裡的 {} 會加強迴避效果
  • prompt 上限約 512 tokens（含角色 prompt）
  • 不支援 Unicode（emoji、日文等）
