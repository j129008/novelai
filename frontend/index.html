Multi-Character

Separate base prompt and each character with |:

base prompt | character 1 description | character 2 description

Base prompt contains:

  • Count: 2girls, 1boy, 1girl
  • Scene/background: park, classroom, sunset
  • Style/mood: watercolor, dramatic lighting, year 2024
  • Composition: from above, wide shot, close-up

Character prompt contains:

  • Use girl/boy only (no number prefix)
  • Appearance: blonde hair, blue eyes, school uniform
  • Expression/action: smile, waving
  • Character name: methode (sousou no frieren)

Full example:

2girls, cherry blossoms, spring, sunny, watercolor | girl, blonde hair, white dress, smile, waving | girl, black hair, blue uniform, holding book, calm

Interactions: source#hug (initiator), target#hug (receiver), mutual#holding hands (both)

Order = position: first character on the left, second on the right, and so on

Tips

  • Tags earlier in the prompt have stronger influence
  • V4.5 supports natural language — you don't have to use tags exclusively
  • {} in Undesired Content strengthens avoidance
  • Prompt limit is ~512 tokens (including character prompts)
  • Unicode is not supported (emoji, Japanese text, etc.)
